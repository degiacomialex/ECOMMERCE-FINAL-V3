# INSTRUCCIONES DE VISUALIZACIÓN

**ASUNTO:** Problema Crítico de Despliegue (Build Fails / Incompatibilidad de Rutas)

**PROBLEMA:** La versión desplegada en GitHub Pages falla al cargar las imágenes y activos estáticos (error 404/Build Fail) debido a una incompatibilidad de configuración de rutas (`base path`) entre Vite y el servidor.

**SOLUCIÓN (PARA VER EL PROYECTO 100% FUNCIONAL):**

El código funcional está completo. Para ver la aplicación operando correctamente (con imágenes, carrito y filtros), siga estos pasos en su máquina local:

1.  Descargar y descomprimir el repositorio.
2.  Abrir la terminal en la carpeta **`ECOMMERCE-FINAL-V3/client`**.
3.  Ejecutar el comando de instalación: **`npm install`**
4.  Ejecutar el comando de desarrollo: **`npm run dev`**

**Resultado:** El sitio cargará en su navegador local (ej: http://localhost:5173), donde todas las imágenes cargarán correctamente. Toda la funcionalidad (filtros, carrito, navegación) está completa.
